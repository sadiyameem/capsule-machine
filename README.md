# Capsule Machine

# Description
- This is a web-based capsule machine, where users can shake the machine, see what is inside of the machine, turn the handle, and collect capsules containing toys.

# Features 
- Capsule with random toys inside of each
- Machine containing the capsules
- 360 Handle
- Shake and see inside button

# Tools
- HTML
- CSS
- JavaScript

# Links
- https://htmlcolorcodes.com/

# Physics Concept
- Gravity pulls the capsules down using acceleration
- Velocity controls how fast and the directions each capsule moves in
- Friction slows the capsules down over time
- Collision checks when capsules hit the walls or each other
- Rotation and motion makes the capsules spin as they move
